JH vs SWS

Goal

Look at SWS and JH differentiation for a series of outcomes (K6, PSS, EDS sum, and EDS chronicity)

List of confounders

  • education (educ_cat, effect coded 0 (less than HS diploma) and 1 (at least high school diploma))
  • marital status (marstatus_dichot, effect coded 0 (not married), 1 (married, living, separated))
  • household income (hshldinc)
  • employment status (employ_dichot, effect coded 0 (unemployed) and 1 (employed) )
  • mibi pub reg (mibi_pubreg)
  • mibi centrality (mibi_central)
  • age (age_yrs)

All SEM paths will be the same as pictured when swapping out DVs with the exception of noted differences for K6 and PSS: K6 & PSS not including age or mibi-centrality.

Running the Path Model

model <- "
# measurement model

# regressions
eds_SUM ~ JH_sum + SWS_careforothers+ SWS_emotion +SWS_strength + SWS_succeed + SWS_vulnerable + educ_cat + employ_dichot +marstatus_dichot +hshldinc + mibi_pubreg + age_yrs +mibi_central

eds_chron_sum ~ JH_sum + SWS_careforothers+ SWS_emotion +SWS_strength + SWS_succeed + SWS_vulnerable + educ_cat + employ_dichot +marstatus_dichot +hshldinc + mibi_pubreg + age_yrs +mibi_central

kesslerSUM ~ JH_sum + SWS_emotion +SWS_strength + SWS_succeed + SWS_vulnerable+ educ_cat + employ_dichot +marstatus_dichot +hshldinc + mibi_pubreg

pss_SUM ~ JH_sum + SWS_emotion +SWS_strength + SWS_succeed + SWS_vulnerable+ educ_cat + employ_dichot +marstatus_dichot +hshldinc + mibi_pubreg
"
fit <- lavaan::sem(model, data = dataset, fixed.x = FALSE)
summary(fit, standardized = TRUE)

Summary of Findings

EDS Sum vs Chronicity

Results flipped when going from the sum of EDS versus the chronicity version of EDS.

hist(dataset$eds_SUM/10)

hist(dataset$eds_chron_sum)

EDS Sum

  • JH Negatively predicts EDS (p = 0.019)
  • SWS Care for others Positively predicts EDS (p = 0.008)
  • SWS Strength Positively predicts EDS (p = 0.034)

EDS Chronicity

  • JH Positively predicts EDS (p = 0.01)
  • SWS Care for others Negatively predicts EDS (p = 0.028)
  • SWS Emotion marginally Negatively predicts EDS (p = 0.09)
  • SWS Strength Negatively predicts EDS (p = 0.048)

K6

  • JH Negatively predicts K6 (p = 0.0005)
  • SWS Emotion Positively predicts K6 (p = 0.005)

PSS

  • SWS Vulnerable Positively predicts PSS (p = 0.016)
